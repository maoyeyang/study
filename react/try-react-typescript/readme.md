# typescript

- typescript 将 js 弱类型变成强类型

  类型声明 文件 @types/react

- webpack-dev-server 不会刷新浏览器

- webpack-dev-server --inline 刷新新浏览器

- webpack-dev-server --inline --hot 刷新浏览器需要更新的部分 热更新

- typescript 是 js 的超集,js 是在 ts 里可以完全运行的

  跟 java 一样, 静态类型语言, 先编译一下

- 强制类型声明
